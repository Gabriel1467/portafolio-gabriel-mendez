# Módulo 10 — Seguridad de Redes WiFi

Este trabajo corresponde al módulo de Hacking Ético enfocado en redes WiFi.  
Aquí se incluyen los archivos del trabajo práctico, presentación y evidencias.

### Contenidos aprendidos:
- Análisis de tráfico y captura de paquetes
- Uso de Wireshark para inspección de red
- Escaneo con Nmap
- Automatización de auditorías con Wifite2
- Identificación de vulnerabilidades en redes WiFi
- Buenas prácticas de seguridad inalámbrica

### Archivos incluidos:
- Presentación del módulo
- Evidencias y capturas (si corresponde)

### Objetivo del trabajo:
Comprender las vulnerabilidades más comunes en redes WiFi y aprender a realizar auditorías básicas de seguridad bajo un enfoque de hacking ético.
